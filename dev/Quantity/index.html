Class: Quantity

A structured object to represent an amount of something (e.g., weight, mass, length, duration of time) - including a value and unit.

URI: ccdh:Quantity

Parents

  • is_a: Entity - Any resource that has its own identifier

Attributes

Own

  • ➞comparator OPT
    • Description: How to understand the value (e.g. < | <= | = | >= | > )
    • range: String
  • ➞unit OPT
    • Description: The unit of the amount represented by the value (e.g. mg, mL, days). This may be a free-text form of the unitCode.
    • range: String
  • ➞unitCode OPT
    • Description: A coded representation of the unit.
    • range: String
  • ➞value REQ
    • Description: An amount, in the given units (if specified)
    • range: String

Inherited from Entity:

  • ➞id OPT
    • Description: The 'logical' identifier of the entity in the system of record, e.g. a UUID. This 'id' is unique within a given system of record. The identified entity (with same "identifier" value for example) may have a different 'id' in a different system.
    • range: String
